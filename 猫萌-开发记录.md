## 配置文件 `config.json`

| 字段 | 说明 |
|---|---|
| `napcat_ws` | NapCat WebSocket 地址，默认 `ws://127.0.0.1:3001` |
| `token` | NapCat 后台的访问令牌（当前：`yTNmnrQurANcU06U`） |
| `web_port` | 本地网页端口，默认 `8080` |
| `buffer_minutes` | 消息缓冲窗口（分钟），默认 `2` |
| `monitored` | `"all"` 全监控，或 `[群号, QQ号]` 指定 |

## 功能清单

- **实时捕获** 所有群聊 + 私聊消息，缓存 2 分钟
- **撤回检测** 触发时立即保存，并推送 SSE 通知到页面
- **媒体下载** 消息到达时自动下载图片/视频/语音到 `data/media/`
- **Web 界面**（猫萌）
  - 左侧栏：按群/好友分类，显示未读红点
  - 点击群/好友：全部标已读（同 QQ 逻辑）
  - 消息卡片：未读时左侧红线标注，点开后平滑消除
  - 详情弹窗：文字/图片/视频/语音全部渲染，视频附下载链接
  - SSE 实时推送：有新撤回时自动刷新，标签页标题显示未读数

## 已知问题 / 待确认

- **私聊撤回** 理论上支持，实际效果待测试（NapCat 版本差异）
- **QQ 语音**（`.silk` 格式）浏览器无法原生播放，页面提供下载链接
- **视频播放** 若浏览器解码失败，页面也提供下载链接兜底
